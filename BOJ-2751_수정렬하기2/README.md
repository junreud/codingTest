# BOJ 2751 - 수 정렬하기 2

## 문제 개요
N개의 수가 주어졌을 때, 이를 오름차순으로 정렬하는 프로그램을 작성하는 문제

## 핵심 포인트
1. **시간 복잡도**: O(N log N) 알고리즘 사용 필요
2. **빠른 입출력**: `sys.stdin.readline` 사용으로 입출력 시간 단축
3. **정렬 알고리즘**: Python의 내장 `sort()` 또는 `sorted()` 함수 활용

## 주요 학습 내용
- 시간 복잡도의 중요성
- Python에서의 빠른 입출력 방법
- 내장 정렬 함수의 효율성
