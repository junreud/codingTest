## 💡 주요 학습 포인트

### 자주 사용하는 Python 패턴
```python
# 빠른 입출력
import sys
input = sys.stdin.readline

# Set을 이용한 빠른 검색
exists = num in number_set  # O(1)

# 정렬과 함께 사용하는 key 함수

# 좌표 압축
coord_map = {val: idx for idx, val in enumerate(sorted(set(coords)))}
```

### 시간복잡도 최적화
- 정렬: O(N log N)
- Set 검색: O(1)
- 그리디: O(N) 또는 O(N log N)
